1.    Description

It is an application that displays the photo (jpg, png). Photos to be displayed will find photos from the internal storage Pictures and DCIM. However, if the SDCARD is mounted to find the photos from the entire SDCARD in addition to the internal storage. Search folder can be changed from the Settings menu.

        Folder list

Is a list of the directory where the photo files are present, display order can be specified in ascending or descending order in any of the folder name and the folder update time. The initial value is the ascending order of the folder name. In addition, you can select a folder in the first letter of the folder name.

①     Exclude

To exclude from the display of the selected folder.

②     Delete

All the photos that are included in the selected folder and delete it.

③     Rename

Rename the folder.

 

        Picture Thumbnail

Display order of photo file name, can be specified in ascending or descending order in any of the shooting time and the file update time. The initial value is descending order of shooting time. (New display from ones) folder you change the display order is maintained even after the application restart. In addition, you can select the photos in the shooting month (yyyy / mm).

①     Send

You can send photos such as mail and Bluetooth.

②     Delete

Delete the selected pictures.

③     Rename

Rename picture file name.

④     Paste

Paste copy or cut items.

⑤     Copy

Copy pictures.

⑥     Cut

Cut the picture. (Picture will be deleted after paste)

 

        Picture screen

①     Full screen

Switching the display and non-display of the operation buttons with a single tap.

②     Photograph information

Show Photographing date, Aperture, Shutter speed, Exposure bias, ISO speed rate, Exposure mode and Camera model name. You can switch the display at the touch of a button.

When the operation button is displayed, you can zoom in button.

        Zoom with a double tap, zoom back to 100% Once you become a 800%

        Zoom and zoom out in a pinch

③     Lock zoom ratio

To display the next or previous photo in the zoom ratio when you press the zoom lock button.

④     Rotate picture

You can 90 degrees right rotation and left rotation.

⑤     Lock screen orientation

It is fixed in the direction of the screen when you press the lock button. However, this function can not be used if the orientation of the screen is fixed in the system settings.

⑥     Show MAP

If the GPS information is included can MAP display.

⑦     Wall paper

To set the wallpaper

⑧     Delete

To delete picture

 

        Menu

①     Refresh file list

Re-read the file information. When running the app, please run when you add or delete photos in the camera or a file manager. (It will be automatically executed when the application restart)

②     Edit picture folder

Edit the folder where you want to search for photos. The initial value will find photos from the internal storage Pictures and DCIM. However, if the SDCARD is mounted to find the photos from the entire SDCARD in addition to the internal storage.

③     Show hidden directory/Do not show hidden directory

Shows or hides the files and hidden files that are included in a hidden directory.

④     Recreate cache file

Delete the file and cache for work. Reading the file information will take about 40 seconds in the 7000 file.

⑤     Manage log file

Send or delete the log.

⑥     Settings

        Hidden file

Shows or hides the files and hidden files that are included in a hidden directory.

        Folder filter character count

You create a filter in the number of characters from the beginning of the folder name.

        The display position of the camera folder(DCIM)

Please check if you want to display the camera folder always at the top.

        File change detection

Please check if the app is to reflect on the screen to detect file changes when the transition from the background to the foreground.

        Photo display settings

Please check if you want to restore the settings of the previous photo display screen.

        Picture screen

Please check if you do not want to display the operation buttons when the photo display. It is an operation button for a single tap will display the photo.

        Screen brightness

Please check if you want to the brightness of the screen to the maximum in the photo display at the time.
